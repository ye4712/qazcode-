## Ережеге

QazCode қазақша бағдарламалау тілі болып табылады және төмендегі ережелерге сәйкес жұмыс істейді:

- **Регистр сезімді**: `х` және `Х` - ішіндегі айнымалылар
- **Комментарийлер**: `#` белгісінен басталады
- **Синтаксис**: Идентификаторларда қазақша және ағылшын әрілері қолдану жүргізіледі

---

## Синтаксис

### Бағдарламаның Құрылымы

```qaz
# Коммент - бұл жол орындалмайды
басып_шығару("Мәтін");
х = 10;
```

### Идентификаторлар

Айнымалы атау қоршетіктерінің бірінші таңбасынан басталуы керек:

```qaz
х = 5;           # Ақты
аты = "Аман";    # Ақты
1х = 10;         # Дұрыс емес - сан өндіктемесінен кейін
```

## Ықпал Операторлары

### Басып Шығару

```qaz
басып_шығару("Мәтін");
басып_шығару(х);
басып_шығару(х + у);
```

**Тамбалау**: 
- Мәтіндік мәнділіктер қос немесе бір тырнақта ықтиялын өткізіледі
- Айнымалыларды және өрнектерді құйын ықпал ету арқылы іске қосады

### Шартты Операторлар (егер)

```qaz
егер х > 5:
    басып_шығару("х үлкен");
```

**Синтаксис**:
- `егер` - шартты оператор сөзі
- Шарт - логикалық өрнек
- `:` - шартты аяқтау
- Сәрі жолы ішіндегі işіңіз - сараттамалар

## Айнымалылар

### Айнымалы Түрлері

1. **Сан** (Number)
   ```qaz
   х = 10;
   у = 3.14;
   ```

2. **Мәтін** (String)
   ```qaz
   аты = "Сәлем";
   сәлеме = 'Әлем';
   ```

3. **Логикалық** (Boolean)
   ```qaz
   шындық = 1;     # True
   жалған = 0;     # False
   ```

### Айнымалы Анықтамасы

```qaz
х = 5;              # Сан
у = "Мәтін";        # Мәтін
з = х + у;          # Өрнек
```

## Ифодалар

### Арифметикалық Операторлар

| Оператор | Көрсету | Мысал |
|---------|---------|-------|
| + | Қосу | `х = 5 + 3;` |
| - | Ағысындау | `х = 5 - 3;` |
| * | Көбейту | `х = 5 * 3;` |
| / | Бөлу | `х = 5 / 3;` |

### Салыстыру Операторлары

| Оператор | Теңдеуі | Мысал |
|---------|--------|-------|
| > | Үлкен | `х > 5` |
| < | Аз | `х < 5` |
| == | Теңді | `х == 5` |
| != | Теңді емес | `х != 5` |
| >= | Үлкен немесе теңді | `х >= 5` |
| <= | Аз немесе теңді | `х <= 5` |

### Логикалық Операторлар

| Оператор | Теңдеуі | Мысал |
|---------|--------|-------|
| && | және | `х > 5 && у < 10` |
| \|\| | немесе | `х > 5 \|\| у < 10` |
| ! | емес | `!(х > 5)` |

шындық = 1;
жалған = 0;

## Мысалдар

### Мысал 1: Әрі Басқарысы Сәрісі

```qaz
х = 10;
у = 20;
з = х + у;
басып_шығару(з);
```

**Нәтиже**: `30`

### Мысал 2: Шартты Сәрісі

```qaz
балл = 85;
егер балл >= 80: басып_шығару("Өте жақсы!");
```

**Нәтиже**: `Өте жақсы!`

### Мысал 3: Мәтіндік Операторлары

```qaz
аты = "Аман";
аттамасы = " Сәлем";
басып_шығару(аты + аттамасы);
```

**Нәтиже**: `Аман Сәлем`
